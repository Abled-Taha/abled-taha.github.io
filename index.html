This project involves the creation of a web application with a multi-faceted approach. It has 5 main components.

  1. Front-End: The user interface is developed using HTML and CSS, ensuring an attractive and user-friendly design to enhance user experience.
  2. Back-End: The Django framework in Python is utilized for server-side functionalities, handling data processing and business logic efficiently.
  3. DB: MongoDB is integrated as the database to store incoming requests for contacting the developer, ensuring scalability and flexibility in managing data.
  4. Mobile App: A mobile application is developed using the Flutter library of the Dart Language. This app provides developers with a convenient means to view and manage incoming requests directly on their mobile devices, enhancing accessibility and responsiveness.
  5. API: The API for this project is completely separate and is made using the Flask framework of the Python language. It serves as the bridge between the front-end interface and the back-end logic, facilitating seamless communication and data exchange.
In summary, this project not only demonstrates proficiency in web technologies but also emphasizes a holistic solution by extending functionality to a mobile platform, thereby enhancing accessibility and user engagement.

This Project is a Password Manager called APM. It has 3 main components.

  1. App: The password manager app is developed using Flutter, offering cross-platform compatibility for Windows, Linux, and Android users. Additionally, it is accessible via a website, ensuring convenient access across different devices and operating systems.
  2. API: The API, built with Django, serves as the backend infrastructure for the password manager system. It provides the necessary functionality for managing user accounts, storing encrypted passwords, and facilitating communication between the app and the database.
  3. DB: MongoDB serves as the database for the password manager system. It stores user credentials and other relevant information securely, ensuring data integrity and confidentiality.
In summary, this project offers users a robust and secure solution for managing their passwords across multiple platforms, with an intuitive app interface, a reliable backend API, and a scalable database infrastructure.
